## ETS2 Mini Companion for SimHub

### Widgets
- Current Speed & Gear
- Speed Limit
- Cruise Control information
- Fuel & Sleep deltas
- Destination estimations
- RPM Gauge
- Steering Gauge
- Light modes & Indicators

### Features
- Automatic speed unit detection
- Warning animations (fuel, rest, speed)
